# Task 3: Linear Regression

## Objective
Implement simple and multiple linear regression using Scikit-learn and evaluate the model.

## Dataset
House Price Prediction Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps Performed

- Preprocessed data

- Trained Linear Regression model
- Evaluated using MAE, MSE and R² Score
- Plotted Actual vs Predicted values
- Interpreted regression coefficients

- Performance Metrics
